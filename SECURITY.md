# Security Policy

## Supported MC Versions

I will not do backports for any reason.

If you have an issue in a version that is no longer supported, backport the latest version yourself and make a pull request.

## Reporting a Vulnerability

DM me on discord (@Omena0), with an explanation.

The scripts having system access is not a vulnerability. It is intended and will not be changed. I will not add sandboxing.
